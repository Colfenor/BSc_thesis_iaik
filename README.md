# BSc_thesis_iaik

Bachelor thesis "Classic McEliece in Rust"


How to build pdf
----------------

You can use `make` to build `thesis.pdf` and `make clean` to remove temporary files from the build process. 


Requires the packages:
- `latexmk`
- `biber`
- `texlive-binextra`
- `texlive-langgerman`
- `texlive-plaingeneric`
- `texlive-bibtexextra`
